# Case Material Desk

ケース面接で使える無料ニュース・企業公式資料・公的情報を整理する材料管理アプリです。

## 方針

- 就活生が無料で読める記事を中心に集めます。
- 有料ニュースAPI、OpenAI API、X APIは使っていません。
- RSSとGoogle News RSSから記事候補を取得します。
- 「無料記事を探す」を押すと、業界・ビジネス・オペレーション・公共情報を中心に、毎回少し違う記事候補を探します。
- 同じテーマの関連記事は、企業・業界・論点が近いものだけを表示します。
- 各記事に「なぜ重要か」「就活・ケース面接で使える理由」を自動表示します。

## 使い方

1. `index.html` を開きます。
2. 「無料記事を探す」を押します。
3. 表示された記事から気になるものを選びます。
4. 気になる記事に、要約・自分の意見・ケースで使えそうな切り口をメモします。
5. 必要なものは「材料として保存」でストックします。

## 公開方法

Vercelで公開する場合、このフォルダの中身をGitHubにアップロードします。

アップロードするフォルダ:

`outputs/job-news-desk`

Vercelでは `api/articles.js` がRSS取得の中継役になります。これは有料APIではありません。

## データ保存

メモや保存材料はブラウザ内の `localStorage` に保存されます。端末やブラウザを変えると見えなくなるため、必要に応じて「全データを書き出し」でバックアップしてください。

## 注意

RSSに載っている範囲の記事だけを取得します。すべての記事を網羅するものではありません。

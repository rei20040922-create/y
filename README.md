# Case Material Desk

ケース面接で使える無料記事・企業公式資料・公的情報を整理する材料管理アプリです。

## 方針

無料記事候補はアプリ側で取得できます。
それに加えて、ユーザーが見つけた企業公式資料や公的情報も手動登録できます。

## 使い方

`index.html` をブラウザで開けば使えます。

## できること

- 無料記事・企業公式資料・公的情報を手動登録
- 無料記事候補を取得
- 同じテーマの関連記事をリンク付きで表示
- テーマ、材料タイプ、企業名、見たい角度を管理
- 記事ごとに要約・自分の意見・ケース切り口を保存
- ケースで使える切り口をテンプレート生成
- 材料ストックに保存
- 検索・絞り込み
- 全データのバックアップ・復元
- スマホ表示対応

## データ保存

データはブラウザ内の `localStorage` に保存されます。
端末やブラウザを変える場合は、アプリ内の「全データを書き出し」でバックアップしてください。

## 本番公開時の注意

このアプリは基本的に静的ファイルで動きます。
無料記事候補の取得には、同梱しているサーバーレス関数を使います。
ログインやクラウドDBはまだありません。
複数端末で同じデータを共有したい場合は、次の段階でDB保存を追加します。

## デプロイ

静的サイトとしてそのまま公開できます。

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

公開する場合は、この `job-news-desk` フォルダの中身をアップロードします。
Vercelでは `api/articles.js`、Netlifyでは `netlify/functions/articles.js` が無料記事候補の取得に使われます。

PWAのオフラインキャッシュは、HTTPSで公開した場合に有効になります。

## 課金

OpenAI API、X API、有料ニュースAPI、有料クラウドDBは未使用です。
無料RSSとGoogle News RSSを使って記事候補を取得します。
